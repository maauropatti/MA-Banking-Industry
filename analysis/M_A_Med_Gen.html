Analisi dei Multipli e della Redditività nel Settore Bancario¶

Wealth Managers vs Banche Tradizionali¶

In questa analisi cerchiamo di individuare una correlazione tra la redditività delle banche, misurata tramite il ROTE (Return on Tangible Equity),, rispetto ai principali multipli di valutazione come il P/B (Price-to-Book) e il P/E (Price-to-Earnings).

L'obiettivo è comprendere se esiste un gap di valutazione sistematico tra le banche tradizionali e i gestori di risparmio, e in che misura tale gap sia giustificato da differenze strutturali in termini di redditività o crescita prospettica.

L’analisi si basa sui dati finanziari delle principali banche italiane nel periodo 2022–2027.

ROTE vs P/E¶

Regressione lineare e confronto tra gruppi bancari¶

In questo grafico analizziamo la relazione tra la redditività delle banche (misurata dal ROTE, asse X) e i multipli di valutazione P/E (asse Y).
È stata applicata una regressione lineare per stimare il legame tra le due variabili: in particolare, se e quanto un aumento della redditività sia associato a un incremento del multiplo P/E.

I punti sono colorati in base al gruppo di appartenenza della banca:

  • Banche tradizionali: Unicredit, Banco BPM, Intesa Sanpaolo
  • Gestori di risparmio: Fineco, Mediolanum, Banca Generali
  • Mediobanca è rappresentata separatamente per distinguerla come caso intermedio

La linea tratteggiata nera rappresenta la retta di regressione calcolata sui dati di tutte le banche.
Il confronto visivo dei gruppi permette di identificare differenze sistematiche nei multipli: ad esempio, se i gestori tendono a posizionarsi sopra la retta (cioè con un P/E più alto rispetto a quanto il ROTE spiegherebbe), questo può indicare un premio di valutazione strutturale.

L'obiettivo è evidenziare possibili gap di valutazione tra categorie bancarie, che non siano giustificati solo dalla redditività, ma anche da aspettative di crescita, qualità degli utili o modello di business.

import matplotlib.pyplot as plt
import numpy as np
from matplotlib.patches import Patch

# Estraggo dati
x = df.iloc[:, 1]  # ROTE
y = df.iloc[:, 3]  # P/E
nomi = df.iloc[:, 0]  # Nomi banche

# Definizione dei gruppi
tradizionali = ["UniCredit", "Banco BPM", "Intesa Sanpaolo"]
gestori = ["FinecoBank", "Banca Mediolanum", "Banca Generali"]
mediobanca = ["Mediobanca"]

# Colori personalizzati
colori = []
for nome in nomi:
    if nome in tradizionali:
        colori.append("#1f77b4")  # blu navy
    elif nome in gestori:
        colori.append("#ff7f0e")  # arancione
    elif nome in mediobanca:
        colori.append("#d62728")  # rosso scuro
    else:
        colori.append("#7f7f7f")  # grigio neutro

# Creo il grafico
plt.figure(figsize=(10, 7))
plt.scatter(x, y, c=colori, edgecolors='black', s=100, alpha=0.8, linewidth=0.7)

# Regressione lineare
m, b = np.polyfit(x, y, 1)
plt.plot(x, m*x + b, color="black", linestyle="--", linewidth=2.0, label=f'Regressione: y = {m:.2f}x + {b:.2f}')

# Etichette
plt.title("P/E vs ROTE: confronto tra gruppi bancari", fontsize=16)
plt.xlabel("ROTE (%)", fontsize=14)
plt.ylabel("P/E Ratio", fontsize=14)
plt.grid(True, linestyle='--', alpha=0.6)

# Legenda
legend_elements = [
    Patch(facecolor='#1f77b4', edgecolor='black', label='Banche tradizionali'),
    Patch(facecolor='#ff7f0e', edgecolor='black', label='Gestori di risparmio'),
    Patch(facecolor='#d62728', edgecolor='black', label='Mediobanca'),
    Patch(facecolor='black', label='Regressione lineare', linestyle='--')
]
plt.legend(handles=legend_elements, fontsize=12, loc='upper left', frameon=True)

plt.tight_layout()
plt.show()

# 1. Estraggo le variabili dai dati originali
x = df.iloc[:, 1]  # ROTE
y = df.iloc[:, 3]  # P/E

# 2. Calcolo regressione lineare su tutti i dati
m, b = np.polyfit(x, y, 1)
pe_atteso = m * x + b



# 4. Aggiungo colonne calcolate al dataframe originale
df["P/E atteso"] = pe_atteso
df["Gap valutazione"] = df.iloc[:, 3] - df["P/E atteso"]

# 5. Raggruppa per banca (colonna 0), calcolando le medie
df_media_pe = df.groupby(df.columns[0]).agg({
    df.columns[1]: 'mean',     # ROTE medio
    df.columns[3]: 'mean',     # P/E medio
    "P/E atteso": 'mean',      # P/E atteso medio
    "Gap valutazione": 'mean'  # Gap medio
}).reset_index()

# 6. Rinomino colonne e ordina per gap
df_media_pe.columns = ["Banca", "ROTE medio", "P/E medio", "P/E atteso", "Gap valutazione"]
df_gap_pe = df_media_pe.sort_values(by="Gap valutazione", ascending=False)

# 7. Mostr0 la tabella finale
df_gap_pe

Analisi della regressione P/E vs ROTE¶

Modello OLS¶

  • Variabile dipendente: P/E
  • Numero di osservazioni: 42
  • R² = 0.456 → il modello spiega circa il 45.6% della varianza del P/E
  • Adj. R² = 0.443
  • F-statistic = 33.57, p-value = 9.21e-07 → modello altamente significativo

Coefficienti del modello¶

  • ROTE è un predittore altamente significativo del P/E
  • Ogni aumento di 1 punto percentuale di ROTE è associato a un aumento atteso di circa 0.36 punti nel P/E
  • Anche l’intercetta è significativa

Diagnostica¶

  • Durbin-Watson = 1.202 → possibile autocorrelazione positiva dei residui
  • Omnibus = 17.313, Prob(Omnibus) = 0.000
  • Jarque-Bera = 27.876, Prob(JB) = 8.85e-07 → residui non normali
  • Skew = 1.131, Kurtosis = 6.289 → asimmetria e code pesanti

Conclusione¶

Il modello è statisticamente significativo e mostra un forte legame tra ROTE e P/E. Tuttavia, l’analisi dei residui evidenzia deviazioni dalla normalità e possibili autocorrelazioni. La capacità esplicativa è moderata, ma il modello è già utile per stimare valutazioni relative.

Analisi della regressione P/BV vs ROTE¶

Modello OLS¶

  • Variabile dipendente: P/BV (Price-to-Book Value)

Conclusione¶

Il modello è robusto, significativo e ben calibrato. Il ROTE è un forte predittore del P/BV, con una capacità esplicativa elevata. È uno strumento efficace per stimare la valutazione di mercato delle banche in funzione della loro redditività.
